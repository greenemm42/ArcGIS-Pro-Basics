# ArcGIS Pro Basics — Course Work

This repository contains personal notes, scripts, project files, and exercises completed while working through the **ArcGIS Pro Basics** course.

## Folder Structure

- **src/** — PowerShell, Python, or tool automation scripts  
- **data/** — raw and processed data used in exercises  
- **docs/** — notes, course summaries, screenshots  
- **output/** — exported maps, reports, logs  

## Workflow

1. Complete a lesson section  
2. Save any scripts to *src/*  
3. Export maps or screenshots into *output/*  
4. Document what you learned in *docs/*  
5. Use Git to commit and push progress
