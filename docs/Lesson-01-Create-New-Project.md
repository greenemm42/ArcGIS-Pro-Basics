# Lesson 01 — Create a New Project

## Objectives
- Learn how to start ArcGIS Pro and create a new map-based project.
- Understand ArcGIS Pro project structure and default folders.
- Configure a project location to support clean data management.
- Ensure the project is set up to connect to external datasets.

## Steps Taken
1. Launched ArcGIS Pro and signed in using an ArcGIS Online account.
2. Selected **Map** under New Project to create a map-based project.
3. Named the project **NorthridgeQuake**.
4. Set the project location inside the structured course directory.
5. Ensured **Create a folder for this project** was enabled.
6. Created the project and verified default items (Maps, Databases, Toolboxes).

## Outcome
A clean ArcGIS Pro project was created with a professional directory structure, ready for data connection and visualization.
