# Lesson 02 — View 2D and 3D Maps

## Objectives
- Open and explore existing 2D and 3D map documents.
- Understand differences between 2D maps and 3D scenes.
- Identify layers, symbology, and spatial relationships.
- Practice navigating maps using ArcGIS Pro tools.

## Steps Taken
1. Downloaded and extracted the View2D3DMaps data into the raw data directory.
2. Connected the NorthridgeData folder to the project using the Catalog pane.
3. Opened the existing 2D map file (NorthridgeQuake).
4. Reviewed layers including earthquakes, faults, roads, building damage, and elevation.
5. Navigated the map using pan and zoom tools.
6. Captured a screenshot of the 2D map.

## Output
- output/screenshots/Northridge_2D_Map.png

## Outcome
Successfully explored earthquake damage data in a 2D map view and documented results with a saved screenshot.
